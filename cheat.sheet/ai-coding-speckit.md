# speckit

<<https://developer.microsoft.com/blog/spec-driven-development-spec-kit*>

## Izolace

Všechno děláš uvnitř kontejneru.

## Scaffold the project

```bash
uvx --from git+https://github.com/github/spec-kit.git specify init <PROJECT_NAME>
```

## slash commands

> 1. Go to the project folder: cd dbt-replacement
> 2. Start using slash commands with your AI agent:<br>
>   2.1 /speckit.constitution - Establish project principles<br>
>   2.2 /speckit.specify - Create baseline specification<br>
>   2.3 /speckit.plan - Create implementation plan<br>
>   2.4 /speckit.tasks - Generate actionable tasks<br>
>   2.5 /speckit.implement - Execute implementation
